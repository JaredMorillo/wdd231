Site Name

Name: Online Board Games Project

Reason for the name: An informative platform about board games that can be played online.

Optional domain: onlineboardgamesproject.net

Site Purpose

This website will provide detailed information about board games that are available in online versions. The site will include:

  • A home page introducing the topic.
  • A board games list page with key details like number of players, playtime, and complexity.
  • A page that includes a form allowing users to interact with the site.
  • A subscription method that lets users connect with others and schedule time to play.

Scenarios

  • Which board games can I play online with my friends, and how many players do they support?
  • How can I subscribe to connect with other players?

Color Scheme

Primary color: #4B79A1 used in headings, borders, and important elements.

Secondary color: #C3D8F2 used as the main background color.

Accent color: #F2E9D8 used for decorative sections and wireframe backgrounds.

Typography

Poppins: used for headings (h1, h2, h3).

Roboto: used for paragraphs and general content.
